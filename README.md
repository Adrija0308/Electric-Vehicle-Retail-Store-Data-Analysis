# Electric Vehicle Retail Store Data Analysis

## Project Overview
This project presents a data-driven analysis of a retail store lead-to-delivery funnel. It focuses on evaluating key performance indicators (KPIs) such as lead conversion, booking efficiency, delivery timelines, cancellations, and customer satisfaction (NPS).

The goal is to identify operational bottlenecks and provide actionable recommendations to improve overall efficiency and customer experience.

---

## Objectives
- Analyze the lead-to-delivery funnel across retail stores  
- Identify drop-offs in conversion stages  
- Evaluate delivery turnaround time (TAT)  
- Understand reasons for customer cancellations  
- Assess customer satisfaction using NPS  
- Perform KPI evaluation to measure overall performance  
- Recommend strategies to improve performance  

---

## Files in Repository
- `store data analysis.xlsx` → Contains cleaned dataset, pivot tables, KPI calculations, and visualizations (charts/plots) used for analysis  
- `Lead-to-Delivery Funnel Analysis Report.pptx` → Final presentation with insights and recommendations  

---

## Key Insights
- Significant drop observed in Lead → Pre-booking stage  
- Warm leads form the majority but show lower conversion efficiency  
- Delivery stage has the highest turnaround time, indicating operational delays  
- Major cancellations driven by pricing and financing issues  
- Customer satisfaction (NPS) is negative across locations, highlighting service gaps  

---

## Methodology
- Data cleaning and preprocessing in Excel  
- Creation of pivot tables to summarize key metrics  
- KPI evaluation (conversion rates, TAT, cancellations, NPS)  
- Visualization of trends using charts and plots  
- Funnel analysis to track conversion across stages  

---

## Recommendations
- Improve follow-up strategy for warm leads (24–48 hr SLA)  
- Introduce automated re-engagement for cold leads  
- Optimize delivery logistics to reduce delays  
- Promote financing/EMI options early in the funnel  
- Implement structured feedback and resolution system to improve NPS
